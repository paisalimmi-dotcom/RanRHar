# RanRHar — บทบาท (Role) เต็มรูปแบบของร้านอาหาร

**Version:** 1.1  
**วันที่:** 2026-02-08  
**ระบบ:** RanRHar Restaurant POS

---

## สรุปภาพรวม

บทบาทแบ่งเป็น **4 กลุ่มหลัก** ตามหน้าที่การทำงาน:

| กลุ่ม | บทบาทในระบบ | บทบาทเสริม (ใช้อธิบายงาน) | หมายเหตุ |
|-------|--------------|----------------------------|----------|
| **ฝ่ายบริหาร** | Owner, Manager | - | กำหนดนโยบายและจัดการร้าน |
| **หน้าร้าน (FOH)** | Staff, Cashier, Host | Bartender | บริการลูกค้า รับชำระ ต้อนรับ |
| **ครัว (BOH)** | Chef | Sous Chef, Line Cook, Prep Cook | ปรุงอาหารและเตรียมวัตถุดิบ |
| **สนับสนุน** | Delivery | Dishwasher, Accountant, System Admin | งานเสริมและการสนับสนุน |

## 9. แผนการใช้บทบาทตามขนาดร้าน

| ขนาดร้าน | บทบาทที่แนะนำ |
|----------|----------------|
| **ร้านเล็ก (1–10 ที่นั่ง)** | Owner + Manager (คนเดียว) หรือ Owner + Staff (ทำทุกอย่าง) |
| **ร้านกลาง (10–50 ที่นั่ง)** | Owner + Manager + Staff + Cashier + Chef |
| **ร้านใหญ่ (50+ ที่นั่ง)** | เพิ่ม Host, Delivery, Sous Chef/Line Cook ตามความจำเป็น |
| **ร้านมีบาร์** | เพิ่ม Bartender (ใช้ staff หรือสร้าง role แยก) |
| **ร้านส่งอาหาร** | เพิ่ม Delivery |
